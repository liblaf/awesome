# `awesome`

**Usage**:

```console
$ awesome [OPTIONS] COMMAND [ARGS]...
```

**Options**:

- `--help`: Show this message and exit.

**Commands**:

- `color`
- `github`
- `website`

## `awesome color`

**Usage**:

```console
$ awesome color [OPTIONS]
```

**Options**:

- `--data PATH`
- `--format [latex|markdown|scss]`: [default: markdown]
- `--help`: Show this message and exit.
